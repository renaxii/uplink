# Uplink

Uplink is an educational satellite cybersecurity game built for the Congressional App Challenge, where the player is protecting satellites and other space infrastructure. 


## Inspiration

Modern society depends on satellites for navigation, communication, weather forecasting, scientific research, and national infrast. As space systems become more important, cybersecurity does as well.

Uplink aims to make satellite cybersecurity more accessible through interactive gameplay and visualization.

## Tech Stack
* Next.js
* TypeScript
* Tailwind CSS
* Three.js
